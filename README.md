# 🚗 DriveEase — Premium Car Rental System

A fully-featured Car Rental Management System built with **JavaFX** for university OOP Lab Project.

## ✨ Features
- 👤 Customer Registration & Login
- 🚗 Browse & Book Vehicles
- 🛒 Shopping Cart System
- 💳 Checkout & Receipt Generation
- 📊 Admin Dashboard with Analytics
- 📋 Booking & Return Management
- 💾 File-based Data Persistence

## 🛠️ Technologies Used
- Java 26
- JavaFX 26
- IntelliJ IDEA

## 🚀 How to Run
1. Install Java 26 from [java.com](https://java.com)
2. Install JavaFX 26 from [gluonhq.com](https://gluonhq.com/products/javafx)
3. Run this command:
```
java --module-path "path/to/javafx/lib" --add-modules javafx.controls,javafx.fxml,javafx.graphics -jar CarRentalSystem.jar
```
